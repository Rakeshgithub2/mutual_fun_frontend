### Local Component State

```typescript
// Simple UI state
```

### Effect Hooks Pattern

```typescript
// Fetch data on mount or dependency change
useEffect(() => {
  const fetchData = async () => {
    setLoading(true);
    try {
      const response = await api.get('/endpoint');
      setData(response.data.data);
    } catch (err) {
      setError(err.message);
    } finally {
      setLoading(false);
    }
  };

  fetchData();
}, [dependency]);

// Debounced search
useEffect(() => {
  const timer = setTimeout(() => {
    if (query) {
      fetchSuggestions(query);
    }
  }, 300);

  return () => clearTimeout(timer);
}, [query]);
```

### Props Drilling vs Context

```typescript
// Props Drilling (Simple)
<Parent>
  <Child data={data} onUpdate={handleUpdate} />
</Parent>;

// Context (Complex/Global)
const AuthContext = createContext();

<AuthProvider>
  <App />
</AuthProvider>;

// In components
const { user, login, logout } = useAuth();
```

## API Endpoints Reference

### Authentication

```
POST   /api/auth/register        - Create account
POST   /api/auth/login           - Login
POST   /api/auth/refresh         - Refresh token
POST   /api/auth/logout          - Logout
GET    /api/auth/google          - Google OAuth
GET    /api/auth/google/callback - OAuth callback
PUT    /api/auth/profile         - Update profile
```

### Funds

```
GET    /api/funds                - List all funds
GET    /api/funds?top=20         - Top 20 funds
GET    /api/funds?category=Equity - Filter by category
GET    /api/funds/{fundId}       - Fund details
GET    /api/suggest?q={query}    - Search suggestions
```

### Comparison

```
POST   /api/comparison/compare   - Compare funds
POST   /api/comparison/overlap   - Calculate overlap
```

### Watchlist

```
GET    /api/watchlist            - Get watchlist
POST   /api/watchlist/{fundId}   - Add to watchlist
DELETE /api/watchlist/{fundId}   - Remove from watchlist
```
